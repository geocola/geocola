A panel container. Works with [tab-container](./tab-container.html) components or as a standalone collapsible panel.
 * @signature `<panel-container />` Example:

 ```html
   <tab-container>
     <panel-container attributes="...">
       <div>content in this panel</div>
     </panel-container>
     <!--additional panels -->
   </tab-container>
 ```
